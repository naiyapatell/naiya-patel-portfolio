# 📍 Campus Lost & Found App

> A systems design project for a mobile lost-and-found platform built to solve a real problem faced by university students.

---

## 📋 Project Overview

**Course:** Systems Design (BADM 353) — Final Project  
**Methodology:** Agile (iterative sprints)  
**Process:** Full Software Development Life Cycle (SDLC)

Many university students lose personal belongings across campus with no reliable, centralized way to recover them. Existing lost-and-found systems are slow, scattered, and offer no real-time updates. This project proposes a mobile application that streamlines the entire lost-and-found process on one secure, accessible platform for students, staff, and local businesses.

---

## 🔍 Systems Design Process

- **Requirements Gathering** — Conducted semi-structured interviews with 8 students and a local business, combined with observational research across campus locations (Union, Main Library, dining halls)
- **User Stories** — Defined functional requirements from the perspective of three user types: students, local businesses, and administrative staff
- **UML Diagrams** — Created Swim Lane diagrams and User Activity diagrams to model system workflows and user interactions
- **Agile Methodology** — Project structured across 4–5 sprints, allowing for iterative development, user feedback, and continuous refinement
- **Prototype** — Developed a functional prototype showcasing core app screens and user flows

---

## ✨ Key Features

- 🔐 **University SSO / Duo Security Authentication** — Secure login tied to university credentials
- 📝 **Lost & Found Item Reporting** — Students and businesses can post items with descriptions, categories, photos, and locations
- 🔎 **Search & Filter** — Filter items by name, category, location, and date
- 📬 **In-App Messaging** — Secure communication between finders and claimants
- ✅ **Claim & Verification System** — Ownership verification workflow before item retrieval
- 🔔 **Push Notifications** — Real-time alerts for matching items and claim status updates
- 🏪 **Business Portal** — Separate login and workflow for local businesses to post and manage found items

---

## 👥 User Types

| User | Role |
|------|------|
| Student | Post lost/found items, search, claim, and message |
| Local Business | Post found items, review claims, communicate with claimants |
| Admin Staff | Oversee posts, manage system integrity |
